void Arrivee::traiter ( )
virtual

Implémente le traitement pour un événement de type 'Arrivee'.

Voir également
Caissier::servir(Client* client)

Cette méthode réalise les actions suivantes :

  • calcul de la prochaine échéance (aléatoire en fonction du temps d'arrivée moyen des clients) et si la prochaine échéance ne dépasse pas la durée prévue :
    • création d'un nouveau client (= arrivée d'un client) ;
    • ajout du client dans la file d'attente selon la disponibilité des caissiers ;
    • création de l'événement et ajout dans la file des événements.

Réimplémentée à partir de Evenement.
